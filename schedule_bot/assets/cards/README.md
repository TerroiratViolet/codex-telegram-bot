# Rider-Waite-Smith Major Arcana images

These 22 images reproduce the original Rider-Waite-Smith Major Arcana
illustrated by Pamela Colman Smith and first published in 1909.

The original deck is in the public domain in the United States and in countries
whose copyright term is no more than 70 years after the death of the last
co-author. Pamela Colman Smith died in 1951, so the original work also entered
the public domain in the United Kingdom at the end of 2021.

The files were retrieved from this pinned public repository revision:

https://github.com/gadicc/magick.ly/tree/f51a84dd1a7177cfb54c20372f0f89a614651a74/public/tarot/rws

That repository records the historical Wikimedia upload URLs in:

https://github.com/gadicc/magick.ly/blob/f51a84dd1a7177cfb54c20372f0f89a614651a74/public/tarot/rws/sources

The images are packaged with this application because those historical
Wikimedia URLs no longer reliably resolve. No modern card back, box design,
publisher branding, or later edition artwork is included.
